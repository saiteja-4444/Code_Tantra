# Understanding Docstring in Python


```python
def add(a, b):
    """Return sum of given arguments."""
    return a + b
def power(b, e):
    """Return the power value.
    
    b -- is the base
    e -- is the exponent
    """
    return b ** e
print(add.__doc__)
    
print(power.__doc__)
```
